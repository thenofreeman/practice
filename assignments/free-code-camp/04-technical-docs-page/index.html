Introduction

The Box is a revolution for the packaging industry, its groundbreaking design and ease of use makes it the perfect choice for your next packaging mission. The Box comes with all the essential needs of a proper packaging basin. Features, such as, but not limited to:

  • Configurable Colors and Materials
  • Study, Yet Lightweight Design
  • Programmable Source Code

Your First Box

Now that we've got all of that out of the way, you're ready to start working with your box. Start by connecting The Box to your computer utilizing the supplied USB-C cable. Download and install the required software and continue on with the next Section.

Configuring Materials

If you haven't already done so, Open the The Box application and click on File > New Box Config. You should be greeted with a breif twenty-five minute, unskippable video tutorial of the application.

Once the video has concluded and you have not filed for a refund within that timeframe; follow the following steps to adjust your boxes materials:

  1. In the top, left hand corner click on the spanner symbol to begin configuration
  2. Select the 'Configure Manually, I'm not stupid' option. A command line will open where in-which we will be spending much of our time
  3. Type in the following code replacing [material-type] with your material of choice and [myBox] with the variable name of your box
    4. java -m [myBox].setMaterialType([material-type]);
    [myBox].type('a').reset();
    followed by
    [myBox].build = true;
    initialize([myBox]);
  4. To specify your desired color type
    5. java -m [myBox].activeModification = true;
    [myBox].setColorAttribute([desired-color]);
    reinitilize([myBox]);
    where [desired-color] is replaced with your chosen color.

Available materials are: 'Metal', 'Wood', 'Cardboard', 'Glass', 'None', 'Mesh', 'Inverted' and 'Organic'

Constructing Your Box

Unplug your newly initilized box from the application, remembering to save before quitting. Fold the bottom carpartments in an secure them with the magic fastening clips.

You are now ready to go!
